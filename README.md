# mcx
MallinCam Xtreme camera control

https://adgsoftware.com/mcx/

### Building

Required packages
```
   cmake
   libwxgtk3.0-dev
   wxformbuilder
```

`wxformbuilder` is optional, only needed if you want to modify the UX.
 
To build:
```
mkdir build
cd build
cmake ..
cmake --build .    # or: make
```
